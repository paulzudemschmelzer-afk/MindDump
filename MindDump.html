Firebase Sync GRATIS

Für Sync zwischen PC und Handy. Einmalige Einrichtung, danach automatisch.

  1. 1
    Geh auf firebase.google.com → "Projekt erstellen"
  2. 2
    Im Projekt: "Web App hinzufügen" → Config kopieren (das JSON-Objekt)
  3. 3
    Firestore Database aktivieren → "Im Testmodus starten"
  4. 4
    Authentication → "Google" als Anmeldeanbieter aktivieren
  5. 5
    Config unten einfügen → Verbinden
Nicht eingerichtet
⚠️ Nach der Einrichtung: In Firebase → Authentication → Settings → "Authorized domains" → deine GitHub Pages URL hinzufügen (z.B. deinname.github.io)

Auf GitHub Pages hosten GRATIS

Damit die App von überall (auch Handy) erreichbar ist.

  1. 1
    Geh auf github.com → Account erstellen → "New repository" → Name: minddump
  2. 2
    Diese HTML-Datei ins Repository hochladen (drag & drop)
  3. 3
    Repository → Settings → Pages → Branch: main → Save
  4. 4
    Deine URL: deinname.github.io/minddump
✓ Dann diese URL auf dem Handy als Lesezeichen oder als Web-App zum Homescreen hinzufügen.
